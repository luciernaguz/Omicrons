About Us

For this project we built a dashboard that displays general information about the NFL teams and their respective stadiums. For our data base we used NFL data APIs from sportsdata-io.com* and we used Flask to connect to our database and create our web application with Python. With it we used bootstrap tools to give some styling to the dashboard and make it more visually appealing, and also Bootstrap is useful for incorporating interactive web pages. D3 helped us give more dynamic features to the page, creating visual representations to data and providing interactive visualizations. And Plotly allowed us to display all the information in statistical charts, graphs, and plots. Finally, we made use of Leaflet to build the map and for making it interactive.
