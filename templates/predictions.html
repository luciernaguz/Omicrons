Overview Analysis

While selecting a topic where we can apply our learnings on Machine Learning, we asked ouserlves if there was a way to create a model that can predict scores based on historical data. Since there are tons of variables to consider, we narrow down our scope to test our model using just a couple of variables and understand the correlation between the variables. So we decided to use to main variables, one is the weather when the game was played and the second one related to location of the game, meaning played as local or a visitor.

Data Source
The data we used for this exercise was pulled from Kraggle. And we use a data set with the NFL results from 1966 season with score games and descriptive info including if a playoff game, played at a neutral site, and weather information if available.

Coding
Coding was done using Python and Jupyter Notebooks, using two main libraries: Pandas for data manipulation and cleanup mainly and Scikit-learn for the predictive model.

4. Build Linear Regression Model
For our model we use the Score as our y value and for our Xs we use the temperature and the local/away game played flag we created. And we train and build our Linear Regression model using sklearn libraries.

5. Results
When running the model we can see the R-squared is pretty low: -0.040. Which tell us the valibles we use do not have a clear correlation with the score.
